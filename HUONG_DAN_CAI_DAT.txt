📋 BƯỚC 1: CÀI ĐẶT PYTHON
═══════════════════════════════════════════════════════════════════════

1. Truy cập: https://www.python.org/downloads/
2. Tải phiên bản Python mới nhất (Python 3.7 trở lên)
3. Chạy file cài đặt vừa tải về
4. ⚠️ QUAN TRỌNG: Tick vào ô "Add Python to PATH" trước khi nhấn Install
5. Nhấn "Install Now" và đợi cài đặt hoàn tất
6. Nhấn "Close" khi hoàn thành

Kiểm tra cài đặt:
- Mở Command Prompt (cmd)
- Gõ: python --version
- Nếu hiện ra phiên bản Python thì đã cài đặt thành công!


📋 BƯỚC 2: CÀI ĐẶT FFMPEG (Tùy chọn nhưng nên cài)
═══════════════════════════════════════════════════════════════════════

FFmpeg giúp tải video chất lượng cao và chuyển đổi sang MP3.

CÁCH 1 - Tự động (Khuyến nghị):
1. Mở Command Prompt với quyền Administrator (nhấn chuột phải -> Run as Administrator)
2. Gõ lệnh: winget install ffmpeg
3. Đợi cài đặt hoàn tất

CÁCH 2 - Thủ công:
1. Tải FFmpeg từ: https://ffmpeg.org/download.html
2. Chọn "Windows builds from gyan.dev"
3. Tải file "ffmpeg-release-essentials.zip"
4. Giải nén vào thư mục (ví dụ: C:\ffmpeg)
5. Thêm vào PATH:
   - Mở "System Properties" -> "Environment Variables"
   - Chọn "Path" trong "System variables"
   - Nhấn "Edit" -> "New"
   - Thêm đường dẫn: C:\ffmpeg\bin
   - Nhấn "OK" để lưu


📋 BƯỚC 3: CÀI ĐẶT SNAPVIDEO
═══════════════════════════════════════════════════════════════════════

1. Giải nén thư mục SnapVideo vào nơi bạn muốn
   (Ví dụ: C:\SnapVideo hoặc Desktop\SnapVideo)

2. Mở Command Prompt trong thư mục SnapVideo:
   - Cách 1: Shift + Chuột phải trong thư mục -> "Open PowerShell window here"
   - Cách 2: Mở cmd và gõ: cd "đường\dẫn\đến\thư\mục\SnapVideo"

3. Cài đặt các thư viện cần thiết, gõ lệnh:
   pip install -r requirements.txt

4. Đợi cài đặt hoàn tất (có thể mất vài phút)

5. Nếu gặp lỗi "pip is not recognized":
   - Thử: python -m pip install -r requirements.txt


📋 BƯỚC 4: CHẠY SNAPVIDEO
═══════════════════════════════════════════════════════════════════════

CÁCH 1 - Double click file .bat (Dễ nhất):
1. Double-click vào file "run_snapvideo.bat"
2. Chương trình sẽ tự động mở!

CÁCH 2 - Chạy bằng Python:
1. Mở Command Prompt trong thư mục SnapVideo
2. Gõ: python snapvideo.py
3. Nhấn Enter


📋 CÁCH SỬ DỤNG
═══════════════════════════════════════════════════════════════════════

1. Mở trình duyệt và truy cập YouTube, Facebook, TikTok, v.v.
2. Copy link của video bạn muốn tải (Ctrl + C)
3. Mở SnapVideo
4. Paste link vào ô "Link Video" (Ctrl + V)
5. Chọn chất lượng video bạn muốn:
   - Tốt nhất: Chất lượng cao nhất có thể
   - 1080p: Full HD
   - 720p: HD
   - 480p: SD
   - 360p: Chất lượng thấp
   - Chỉ âm thanh: Chuyển sang MP3
6. Chọn thư mục lưu file (hoặc để mặc định)
7. Nhấn nút "TẢI VIDEO"
8. Đợi tải xong và thưởng thức!


🐛 XỬ LÝ LỖI THƯỜNG GẶP
═══════════════════════════════════════════════════════════════════════

❌ Lỗi: "Python is not recognized"
✅ Giải pháp: 
   - Cài đặt lại Python
   - Nhớ tick "Add Python to PATH" khi cài đặt

❌ Lỗi: "No module named 'yt_dlp'"
✅ Giải pháp:
   - Chạy: pip install -r requirements.txt
   - Hoặc: python -m pip install -r requirements.txt

❌ Lỗi: "Unable to download video"
✅ Giải pháp:
   - Kiểm tra link video có đúng không
   - Kiểm tra kết nối internet
   - Thử với video khác
   - Video có thể bị giới hạn khu vực

❌ Video không có âm thanh hoặc không tải được 1080p
✅ Giải pháp:
   - Cài đặt FFmpeg (xem Bước 2)
   - Chọn chất lượng thấp hơn (720p hoặc 480p)

❌ Lỗi: "pip is not recognized"
✅ Giải pháp:
   - Thử: python -m pip install -r requirements.txt
   - Hoặc cài đặt lại Python với "Add Python to PATH"


💡 MẸO VÀ TRICKS
