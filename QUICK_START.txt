🚀 PHẦN 1: CHO NGƯỜI DÙNG (Chạy phần mềm)
═══════════════════════════════════════════════════════════════════════

Nếu bạn chỉ muốn SỬ DỤNG SnapVideo:

┌─────────────────────────────────────────────────────────────────┐
│  1. Double-click file: run_snapvideo.bat                        │
│  2. Dán link video vào ô "Link Video"                          │
│  3. Chọn chất lượng                                             │
│  4. Click "TẢI VIDEO"                                           │
│  5. Xong!                                                        │
└─────────────────────────────────────────────────────────────────┘

Đọc thêm: HUONG_DAN_CAI_DAT.txt


🔧 PHẦN 2: CHO DEVELOPER (Build phần mềm)
═══════════════════════════════════════════════════════════════════════

Nếu bạn muốn BUILD file .exe từ source code:

┌─────────────────────────────────────────────────────────────────┐
│  Bước 1: Cài đặt dependencies                                   │
│  >>> pip install -r requirements-dev.txt                        │
│                                                                  │
│  Bước 2: Build file .exe                                        │
│  >>> build_exe.bat                                              │
│  Hoặc: pyinstaller --clean snapvideo.spec                      │
│                                                                  │
│  Bước 3: Kiểm tra                                               │
│  >>> Chạy file SnapVideo.exe                                    │
│                                                                  │
│  Bước 4 (Tùy chọn): Tạo installer                               │
│  >>> build_installer.bat                                        │
└─────────────────────────────────────────────────────────────────┘

Đọc thêm: HUONG_DAN_BUILD.txt

🎯 CÁC FILE BUILD OUTPUT
═══════════════════════════════════════════════════════════════════════

Sau khi build, bạn sẽ có:

SnapVideo.exe                       ← File thực thi độc lập (~30-50MB)
output/SnapVideo_Setup_v1.0.0.exe   ← File installer (~35-55MB)

Cả 2 file đều:
✓ Chạy được trên Windows 7/8/10/11
✓ KHÔNG CẦN cài Python
✓ Có thể chia sẻ cho người khác

⚠️ LƯU Ý QUAN TRỌNG
═══════════════════════════════════════════════════════════════════════

1. Python version: Cần Python 3.7 trở lên
2. FFmpeg: Không bắt buộc nhưng nên cài
3. Internet: Cần để tải video
4. Windows Defender: Có thể cảnh báo lần đầu (bình thường)
5. Build environment: Build trên Windows 64-bit được file 64-bit


🆘 TRỢ GIÚP NHANH
═══════════════════════════════════════════════════════════════════════

❌ Lỗi: "Python is not recognized"
   → Cài Python và tick "Add to PATH"

❌ Lỗi: "No module named..."
   → pip install -r requirements.txt

❌ Lỗi: "Unable to download video"
   → Kiểm tra link, kiểm tra internet

❌ Build lỗi
   → Đọc HUONG_DAN_BUILD.txt

❌ Lỗi khác
   → Đọc README.md
