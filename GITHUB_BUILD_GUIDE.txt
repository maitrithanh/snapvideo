📋 HƯỚNG DẪN TỪNG BƯỚC
═══════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────┐
│ BƯỚC 1: Tạo GitHub Repository                                   │
└─────────────────────────────────────────────────────────────────┘

1. Truy cập: https://github.com/new

2. Điền thông tin:
   - Repository name: snapvideo
   - Description: Video Downloader for Windows
   - Chọn: Public (để được free Actions)
   - Không tick các ô khác

3. Click "Create repository"


┌─────────────────────────────────────────────────────────────────┐
│ BƯỚC 2: Upload code lên GitHub                                  │
└─────────────────────────────────────────────────────────────────┘

CÁCH A - Dùng GitHub Desktop (Dễ nhất):
   1. Download GitHub Desktop: https://desktop.github.com/
   2. Cài đặt và đăng nhập
   3. File → Add Local Repository
   4. Chọn thư mục snapvideo
   5. Click "Publish repository"

CÁCH B - Dùng Terminal/Command Line:
   1. Mở Terminal trong thư mục snapvideo
   2. Chạy các lệnh:
   
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/snapvideo.git
   git push -u origin main
   
   (Thay YOUR_USERNAME bằng username GitHub của bạn)

CÁCH C - Dùng giao diện web GitHub:
   1. Trong repository vừa tạo, click "uploading an existing file"
   2. Kéo thả toàn bộ files vào
   3. Click "Commit changes"


┌─────────────────────────────────────────────────────────────────┐
│ BƯỚC 3: GitHub Actions tự động build!                          │
└─────────────────────────────────────────────────────────────────┘

Sau khi push code lên:

1. Vào repository trên GitHub
2. Click tab "Actions"
3. Bạn sẽ thấy workflow "Build SnapVideo" đang chạy
4. Đợi 3-5 phút để build xong (xem progress bar)
5. Khi xong (màu xanh ✓), click vào workflow run
6. Scroll xuống phần "Artifacts"
7. Click "SnapVideo-Windows" để download file .exe!


┌─────────────────────────────────────────────────────────────────┐
│ BƯỚC 4: Download và sử dụng                                    │
└─────────────────────────────────────────────────────────────────┘

1. Download file zip từ Artifacts
2. Giải nén → có file SnapVideo.exe
3. Double-click để chạy!
4. Chia sẻ file này cho ai bạn muốn!


═══════════════════════════════════════════════════════════════════════
🎯 TẠO RELEASE (Để chia sẻ dễ dàng)
═══════════════════════════════════════════════════════════════════════

Thay vì download từ Artifacts, tạo Release để dễ chia sẻ:

┌─────────────────────────────────────────────────────────────────┐
│ CÁCH 1: Tạo release thủ công                                   │
└─────────────────────────────────────────────────────────────────┘

1. Vào tab "Releases" trong repository
2. Click "Create a new release"
3. Tag version: v1.0.0
4. Title: SnapVideo v1.0.0
5. Describe this release: Mô tả phiên bản
6. Upload file SnapVideo.exe (từ Artifacts đã tải)
7. Click "Publish release"


┌─────────────────────────────────────────────────────────────────┐
│ CÁCH 2: Tạo release tự động bằng tag                           │
└─────────────────────────────────────────────────────────────────┘

Trong Terminal:

git tag v1.0.0
git push origin v1.0.0

→ GitHub Actions sẽ tự động:
  ✓ Build file .exe
  ✓ Tạo release
  ✓ Upload file .exe vào release

Sau đó vào tab Releases để download!


═══════════════════════════════════════════════════════════════════════
🔄 UPDATE PHẦN MỀM
═══════════════════════════════════════════════════════════════════════

Khi bạn sửa code và muốn build lại:

1. Sửa file snapvideo.py (hoặc bất kỳ file nào)

2. Push code mới:
   git add .
   git commit -m "Update features"
   git push

3. GitHub Actions tự động build lại!

4. Download file .exe mới từ Actions → Artifacts


Hoặc tạo version mới:

   git tag v1.0.1
   git push origin v1.0.1

→ Tự động tạo release v1.0.1 với file .exe mới!

═══════════════════════════════════════════════════════════════════════
🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Vấn đề: Actions tab không hiện
→ Vào Settings → Actions → Enable Actions

Vấn đề: Build failed
→ Click vào workflow run để xem log lỗi
→ Thường là thiếu dependencies

Vấn đề: Không thấy Artifacts
→ Đợi build hoàn tất (màu xanh ✓)
→ Scroll xuống cuối trang

Vấn đề: File .exe không download được
→ Phải đăng nhập GitHub
→ Artifacts tự động xóa sau 90 ngày
