📝 TÔI CHỌN: GitHub Actions (Khuyến nghị!)

▶ Làm theo 5 bước:

1️⃣ Tạo GitHub account (nếu chưa có): github.com

2️⃣ Tạo repository mới:
   - Tên: snapvideo
   - Public (để free Actions)

3️⃣ Upload code:
   CÁCH DỄ - Dùng GitHub Desktop:
   • Download: https://desktop.github.com/
   • Add Local Repository → chọn thư mục snapvideo
   • Publish repository

   CÁCH NHANH - Dùng Terminal:
   cd /Users/maitrithanh/Desktop/WorkSpace/snapvideo
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/snapvideo.git
   git push -u origin main

4️⃣ Xem Actions tab → Đợi build xong (3-5 phút)

5️⃣ Download file .exe từ Artifacts!

✅ XONG! Bạn có file SnapVideo.exe!
